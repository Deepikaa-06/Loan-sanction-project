# 🏦 Loan Approval Analysis (EDA Project)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a loan approval dataset. The goal is to understand the data, identify patterns, and generate meaningful insights that can help in understanding factors affecting loan approval.

---

## 🎯 Objectives

* Understand the structure of the dataset
* Handle missing values and clean the data
* Perform univariate, bivariate, and multivariate analysis
* Create attractive visualizations
* Extract meaningful insights from the data

---

## 📂 Dataset Information

The dataset contains details about loan applicants such as:

* Gender
* Marital Status
* Education
* Income (Applicant & Co-applicant)
* Loan Amount
* Loan Term
* Credit History
* Property Area
* Loan Status

---

## 🛠️ Tools & Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Steps Performed

### 1. Data Exploration

* Loaded dataset using Pandas
* Checked dataset shape, columns, and data types
* Identified missing values
* Generated summary statistics

### 2. Data Cleaning

* Filled missing values using:

  * Mode (for categorical data)
  * Median (for numerical data)

### 3. Univariate Analysis

* Histograms for numerical columns
* Boxplots to detect outliers
* Countplots for categorical variables

### 4. Bivariate Analysis

* Scatter plots (Income vs Loan Amount)
* Boxplots (Categorical vs Numerical comparison)

### 5. Multivariate Analysis

* Correlation heatmap
* Stacked bar charts

---

## 📊 Key Insights

* Most applicants have **low to medium income**
* Majority of loans are taken for **long-term duration (360 months)**
* Most applicants have a **good credit history**
* **Male and married applicants** are more common
* Higher income generally leads to **higher loan amounts**
* Loan approvals are higher in **urban and semi-urban areas**

---

## 📈 Conclusion

This analysis helps in understanding applicant behavior and the important factors influencing loan approval. Credit history, income level, and property area play a significant role in determining whether a loan gets approved.
